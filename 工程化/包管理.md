## 包管理

起源： Linux 开源开始
最早先是本地包安装：

-   dpkg （1994）
-   rpm （1997）

没有解决分发问题，于是出现了：远程仓库，集中管理包

-   apt， 管理
-   yum，

基本功能： 安装和依赖管理

现在编程语言的包，不再是二进制的包：

-   perl
-   Java
-   ruby
-   python

都有自己的包管理

语言的包管理工具，并不依赖于语言本身，是一个独立的生态。
